# filmRep
Welcome to film api : (This is coded with .net core and react with Typescript)
### functionality
```
Add movies
Add actors
Sorting
```
### Installing

1.Open cmd and copy this : 
```
git clone https://github.com/martasp/filmRep.git
cd filmRep
dotnet restore
dotnet ef database update
dotnet publish
dotnet run 

```
2.Open browser localhost:5000
