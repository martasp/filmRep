# filmRep
Welcome to film api : (This is coded with .net core and react with Typescript)
### functionality
```
Add movies
Add actors
Sorting
```
### Installing
```
1.Open cmd
2.git clone https://github.com/martasp/filmRep.git
3.cd filmRep
4.dotnet restore
5.dotnet ef database update
6.dotnet publish
7.dotnet run 
8.Open browser localhost:5000
```
